HTML: HyperText Markup Language

The standard language for creating web pages

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and design pages on the World Wide Web. It provides the structure for web content using a system of elements and tags. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript. HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects such as interactive forms may be embedded into the rendered page.

Key Features of HTML

  • Simple and easy to learn
  • Platform independent
  • Supports multimedia (audio, video, images)
  • Integrates with CSS and JavaScript
  • Open standard maintained by W3C

HTML Version History

Version Year Features
HTML 1.0 1993 Basic tags and structure
HTML 4.01 1999 Better layout, CSS support
HTML5 2014 Multimedia, semantic tags
